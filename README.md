### Template Footprint
```bash
# install gastby CLI
npm install -g gatsby-cli

# initial vite-react
gatsby new
> TypeScript
> No CMS ( add later )
> No Additional lib

# initial tailwind
npm install -D tailwindcss postcss autoprefixer gatsby-plugin-postcss
npx tailwindcss init -p
```

### How to run
```bash
# run app local
gatsby develop 
# or 
npm run dev

# production
npm run serve
```
> Server running on =>  http://localhost:8080/
